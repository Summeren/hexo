定时器语法

学习笔记
  • timer
  • xml

日常工作中我们经常会实现这样一些功能，比如定时删除或者下载文件等等。首先我们想到的就是通过一个定时器定时的帮助我们去干一些我们需要它们帮助我们做的事情。

下面我们就了解一下，一些常用的定时器语法。

字段 允许值 允许的特殊符号
秒 0-59 - * /
分 0-59 - * /
小时 0-23 - * /
日期 1-31 - * ? / L W C
月份 1-12或者JAN-DEC - * /
星期 1-7或者SUN-SAT - * ? / L C #
年（可选） 留空，1970-2099 - * /
